# Assets

This directory is reserved for OPL Cloud product maps, architecture diagrams,
and public screenshots.

`branding/opl-cloud-overview.png` is the public README overview image. Deeper
architecture notes may still use Mermaid diagrams directly in Markdown so
GitHub can render them without generated image files.

`branding/opl-gateway-logo-80.png` and `branding/opl-gateway-logo-source.png`
are OPL Gateway brand assets managed under the OPL Cloud public brand surface.
